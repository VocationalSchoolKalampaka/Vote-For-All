# Vote-For-All
Το έργο «Vote For All» υλοποιείται στο Tinkercad και επιτρέπει τη διεξαγωγή ψηφοφορίας στην τάξη με άμεση εμφάνιση αποτελεσμάτων. Χρησιμοποιεί Arduino Uno, κουμπιά, LCD οθόνη και σερβοκινητήρα, συνδυάζοντας προγραμματισμό και ηλεκτρονικά σε ένα λειτουργικό σύστημα. Ξαι φυσικα 3D σχεδίαση και εκτύπωση .
