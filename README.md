# Vote-For-All
Το έργο «Vote For All» υλοποιείται στο Tinkercad και αποτελεί μια καινοτόμα εκπαιδευτική κατασκευή που συνδυάζει προγραμματισμό, ηλεκτρονικά, 3D σχεδίαση και 3D εκτύπωση σε ένα λειτουργικό σύστημα ψηφοφορίας. Οι μαθητές μπορούν να συμμετέχουν σε διαδικασίες λήψης αποφάσεων μέσα στην τάξη, πραγματοποιώντας ψηφοφορίες μέσω κουμπιών εισόδου και διαδραστικών μηχανισμών λειτουργίας. Το σύστημα χρησιμοποιεί Arduino Uno, κουμπιά και σερβοκινητήρα, συνδυάζοντας τεχνολογία και βιωματική μάθηση μέσα από STEM εφαρμογές.

Το έργο δημιουργήθηκε από την ομάδα «Democracy Engineers», η οποία αποτελείται από τον υπεύθυνο καθηγητή Γκέκα Ξενοφώντα και την Ομάδα Ρομποτικής του ΕΠΑΛ Καλαμπάκας. Η συνεργασία μαθητών και εκπαιδευτικού είχε ως στόχο τη δημιουργία ενός σύγχρονου συστήματος που προάγει τη δημοκρατία, τη συνεργασία και τη βιωματική μάθηση μέσα από την τεχνολογία.

Το έργο μας έχει σαν στόχο:

Καλλιέργεια δημοκρατικής κουλτούρας και συμμετοχής
Σύνδεση σχολείου με πραγματικό κόσμο
Ανάπτυξη ενεργητικής και βιωματικής μάθησης
Καλλιέργεια κριτικής και αναλυτικής σκέψης
Ενίσχυση αυτονομίας και υπευθυνότητας
Ανάπτυξη συνεργατικών δεξιοτήτων
Ενίσχυση κινήτρου και ενδιαφέροντος για μάθηση

Θα χρειαστούμε:

Arduino Uno R3 Board
Breadboard Mini Modular
4x Push Button
Micro Servo
Καλώδια
Αντιστάσεις
Νήμα PLA για 3D εκτύπωση
