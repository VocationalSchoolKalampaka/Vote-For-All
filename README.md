# Vote-For-All
Το έργο «Vote For All» υλοποιείται στο Tinkercad και επιτρέπει τη διεξαγωγή ψηφοφορίας στην τάξη με άμεση εμφάνιση αποτελεσμάτων. Χρησιμοποιεί Arduino Uno, κουμπιά, LCD οθόνη και σερβοκινητήρα, συνδυάζοντας προγραμματισμό και ηλεκτρονικά σε ένα λειτουργικό σύστημα. Ξαι φυσικα 3D σχεδίαση και εκτύπωση . 
Το έργο μας έχει σαν στόχο 
1) Καλλιέργεια δημοκρατικής κουλτούρας και συμμετοχής
2) Σύνδεση σχολείου με πραγματικό κόσμο
3) Ανάπτυξη ενεργητικής και βιωματικής μάθησης
4) Καλλιέργεια κριτικής και αναλυτικής σκέψης
5) Ενίσχυση αυτονομίας και υπευθυνότητας
6) Ανάπτυξη συνεργατικών δεξιοτήτων
7) Ενίσχυση κινήτρου και ενδιαφέροντος για μάθηση
Θα χρειαστούμε
1)  Arduino Uno R3 Board
2)  Blue 16x2 LCD Display Module HD44780
3)  Breadboard Mini Modular
4)  4χ Push Button
5)   Micro Servo
6)  καλώδια
7)  αντιστάσεις
8)  νήμα PLA
